What is commerical aviation?

Aviation is anything relating to mechanical flight and the airline industry as a whole. This website specifically will talk about the different aspects of aviation. Aviation can be divided into commerical, military, and private. Commerical aviation can be defined as flights which normal people like you and I would take flights on.
